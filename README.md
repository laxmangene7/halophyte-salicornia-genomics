# halophyte-salicornia-genomics

We assembled six chromosome-scale *Salicornia* genomes and analyzed a global resequencing panel of 318 accessions using comprehensive population genomic analyses, including phylogenetic reconstruction, population structure, nucleotide diversity (π), Tajima's D, genetic differentiation (Weir and Cockerham's FST), read-depth–based genome coverage analysis, and ploidy determination. Our results reveal taxonomic inconsistencies and extensive genetic diversity, establishing a genomic foundation for breeding and the neo-domestication of *Salicornia* for saline agriculture.

## Repository Structure

### Genome-Coverage-Graphs
Contains scripts and input files required to generate genome coverage plots and perform read-depth–based genome coverage analyses.

### Haplotype-Analysis
Contains scripts and input files used for local haplotype analysis presented in **Extended Data Fig. 10b**.

### Phylogeny-PCA
Contains scripts and input files for phylogenetic tree, principal component analysis (PCA), and population structure analysis.

### TajimasD-NucleotideDiversity-Fst
Contains scripts and input files for calculating Tajima's D, nucleotide diversity (π), and Weir and Cockerham's Fst across the target region on chromosome 6A.

### Variant-Calling
Contains scripts and parameter settings used for read alignment, variant calling, variant filtering, and generation of the SNP datasets.
